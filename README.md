## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Peach97/SlickNSpanWebsite
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
   
   #### Start
   
   ```sh
   npm start
   ```

### EmailJS
